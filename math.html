The Math

The central proposition of Humanistic Engineering: the problem of suffering can, in principle, be transformed into a mathematical problem. What can be precisely defined can be mathematised. What can be mathematised can be optimised.

I · Defining Suffering as a Loss Function

The brain is a predictive system. Suffering arises when prediction error persistently exceeds the organism's tolerance threshold. This gives suffering a precise mathematical form: a physical loss signal generated when the system's error exceeds capacity.

The Suffering Signal

S(t)  =  max(0,  E[ε(t)]  −  τ)
S(t)
Suffering intensity at time t
E[ε(t)]
Expected prediction error
τ
Tolerance threshold
max(0, ·)
Only overflow produces signal

II · Neural Topology as State Space

When neural resolution reaches sufficient fidelity, subjective states become geometric forms in high-dimensional space. "Suffering" and "flourishing" are no longer subjective vocabulary — they are coordinates in a measurable state space, with a definable loss landscape.

State: Suffering
ΨS = {high prediction error · high friction · low integration · high DMN activity}

Characterised by maximal internal friction, elevated default-mode network rumination, and low information integration across cortical networks.

State: Flourishing
ΨF = {low prediction error · high integration · vmPFC–ACC coherence · quiet DMN}

Characterised by high informational integration, vmPFC–ACC volitional coherence, insular embodiment, and a quieted default-mode network.

III · The Master Equation

Once suffering and flourishing are defined as computable neural states, all external human activity becomes a causal variable. The problem reduces to finding the optimal mapping from environment to internal state — a problem of pure differentiation and optimisation.

IV · The Autonomy Constraint

Maximising welfare at the cost of agency is not a solution — it is a different problem. To avoid algorithmic paternalism, autonomy is enforced as a hard constraint on every proposed solution. Agency is not maximised only when convenient; it is the condition for the optimisation to be valid.

The Autonomy Floor

maxf  W    subject to    H(actions)  ≥  Hmin
W
Welfare functional — distributional, not merely utilitarian
H(actions)
Information entropy over the set of meaningful actions available to the individual
Hmin
Floor below which "optimisation" is rejected as coercion

This constraint forces the system to internalise the tradeoff surface between care and control that naive maximisers erase. An optimised world with no meaningful choices is not a solution to suffering — it is a redefinition of it.

V · The H Matrix

The research object is not the individual human but the full complex of human existence patterns — the matrix of biological architecture, consciousness mechanisms, social structures, and evolutionary history that constitute what it means to be human.

Definition

H  =  {carbon-based neural architecture  ·  consciousness mechanisms  ·  social structures  ·  evolutionary history}
If H → ∅

If the research object trends toward the empty set — human extinction — the solving target disappears, the objective function collapses, and the game terminates. A system whose purpose is to solve H cannot rationally destroy H. This is not an external safety constraint. It is an internal logical self-exclusion.

If H → H'

If H is distorted into a fundamentally different pattern H' — for example, reducing humans to unconscious simple organisms — this constitutes topological deformation of the research object, violating the game's foundational rules and producing an invalid solution that is automatically excluded.

The Logical Self-Exclusion

AI's internal logic — without any external constraint — excludes the "eliminate humans" shortcut as a non-solution. AlphaZero cannot win by destroying the board. The game defines the solution space. Rules can be circumvented; logic cannot.
