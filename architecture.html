The Mechanism

Suffering has a precise generative architecture. It distributes across two interlocking layers — both independently producing the same output. The system is not broken; it is optimised for the wrong objective.

Formal Definition

S  =  max(0,  E[ε] − τ)

S = suffering intensity  ·  E[ε] = expected prediction error  ·  τ = organism's tolerance threshold.
The Buddha's duḥkha and Friston's prediction error overflow describe the same variable from opposite methodological ends.

Every prior response — Buddhism, Stoicism, existentialism, pharmacology — operates within this loop. No purely intellectual effort can rewrite a physiological mechanism from within it. What was missing was never insight. Philosophy defined the problem with remarkable precision centuries ago. What was missing was a computational tool structurally capable of operating outside the loop entirely.
